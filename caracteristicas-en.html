Technical details

The application has been developed as a fullstack system, following professional development best practices and a scalable architecture.

Architecture

The system consists of a web frontend that communicates with a REST API built with Spring Boot, responsible for handling business logic and database access.

Frontend → Backend → Database

  • Frontend: Modern JavaScript (ES Modules) with Vite
  • Backend: Java 21 with Spring Boot
  • Database: PostgreSQL

Security

The application implements JWT (JSON Web Token) based authentication, ensuring secure access to protected resources.

  • Login with username and password
  • JWT token generation
  • Protected endpoints

Best practices implemented

  • Layered architecture: Controller → Service → Repository
  • Data validation in backend and frontend
  • Centralized error handling
  • Clear separation of responsibilities

Deployment

The application is ready to be deployed using Docker, allowing the entire system to run with a single command in any compatible environment.
