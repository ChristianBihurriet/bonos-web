Características del sistema

Gestión completa de bonos

Permite crear, editar, consultar y eliminar bonos de forma sencilla.

Autenticación segura

Acceso mediante usuario y contraseña con protección JWT.

Control de estados

Bonos clasificados como activo, usado o vencido.

Interfaz clara

Diseño enfocado en la simplicidad y facilidad de uso.

Ventajas

  • Reduce errores manuales
  • Centraliza la información
  • Mejora la organización del negocio
  • Fácil de usar y aprender

Forma de uso

  1. Iniciar sesión
  2. Consultar listado de bonos
  3. Crear o editar bonos
  4. Eliminar bonos si es necesario

Detalles técnicos

La aplicación ha sido desarrollada como un sistema fullstack, siguiendo buenas prácticas de desarrollo profesional y una arquitectura escalable.

Arquitectura

El sistema está compuesto por un frontend web que se comunica con una API REST desarrollada en Spring Boot, encargada de gestionar la lógica de negocio y el acceso a la base de datos.

Frontend → Backend → Base de datos

  • Frontend: JavaScript moderno (ES Modules) con Vite
  • Backend: Java 21 con Spring Boot
  • Base de datos: PostgreSQL

Seguridad

La aplicación implementa un sistema de autenticación basado en JWT (JSON Web Tokens), garantizando el acceso seguro a los recursos protegidos.

  • Inicio de sesión con usuario y contraseña
  • Generación de token JWT
  • Protección de endpoints

Buenas prácticas implementadas

  • Arquitectura en capas: Controller → Service → Repository
  • Validación de datos en backend y frontend
  • Manejo centralizado de errores
  • Separación clara de responsabilidades

Despliegue

La aplicación está preparada para su despliegue mediante Docker, lo que permite ejecutar todo el sistema con un solo comando en cualquier entorno compatible.
